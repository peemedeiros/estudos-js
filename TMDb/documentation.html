Objetivo do projeto

O projeto for desenvolvido para praticar o importante conceito de utilização de API's externas e também introduzir o aluno à prática de consultar as informações e métodos contidos na Documentação da API.

Consumindo API

Entrando na documentacao da API temos varios métodos GET diferentes para trazer os dados, no projeto foi utilizado o método de busca por query em STRING. Logo é possível buscar os filmes com base em uma palavra chave, que será o nome do filme.
Na documentação da API é possível realizar requisições de testes utilizando esse método. Passando como atributos sua chave privada e o nome do filme.

Realize testes

Realizando requisições

Para realizar as consultas utilizando o nome do filme, você deve utilizar a url:

https://api.themoviedb.org/3/search/movie?api_key=APIKEY&language=en-US&query=piratasdocaribe&page=1&include_adult=false

Note que você deve substituir APIKEY por sua chave privada, no atibuto QUERY será o elemento de busca na API, ou seja, é nele em que você deve realizar suas buscas pelo nome do filme.

Atribuindo imagens

Ao observarmos o JSON podemos enxergar que o atributo que contém as imagens é o 'back_drops' porém é necessário dois parâmetros para que as imagens sejam carregadas de forma visível.

São esse atributos:
A URL base https://image.tmdb.org/t/p/
O tamanho da imagem /w500/
E o nome do arquivo, que está guardado no atributo back_drops do JSON

Exemplo: https://image.tmdb.org/t/p/w500/8uO0gUM8aNqYLs1OsTBQiXu0fEv.jpg

NOTA: existem outros parametros para o tamanho da imagem e até mesmo outros métodos de buscar as imagens. Todas disponiveis na documentação oficial da API.
